The RVA Demo Portal consumes cryptographic anchors generated by the Go engine.

### Source
- Default: `internal/merkle/anchor.json` (produced by `go run cmd/rva-rotate/main.go`)
- Override: set environment variable `RVA_ANCHOR_PATH=/path/to/anchor.json`

### Injection
- Run `npm run inject-anchor --prefix rva-demo-portal`
- This executes `scripts/inject-anchor.ts`
- Output: `rva-demo-portal/public/anchor.json`

### Usage
- The DemoClient UI provides a **Load Anchor** button.
- Clicking it fetches `public/anchor.json` and populates the verification form.
- Then, **Execute Verification** runs the proof check against the expected Merkle root.

### Build
- Run `npm run build --prefix rva-demo-portal`
- Produces a static dossier in `rva-demo-portal/out` with the anchor preloaded.
- This artifact is uploaded automatically by the GitHub Actions pipeline (`sovereign-audit.yml`).
